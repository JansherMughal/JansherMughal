> I'm an **AI/ML Engineer & MLOps practitioner** passionate about taking machine learning
> systems from prototype to **production at scale**. At **[Folio3](https://www.folio3.com/)**,
> I build reliable, observable, cloud-native ML platforms that power real products.

**What I do day-to-day:**

- 🚀 Design and maintain **cloud-native ML infrastructure** on AWS (ECS, Fargate, Terraform)
- 🤖 Ship **LLM-powered applications** with RAG, vector databases, and modern AI tooling
- 👁️ Engineer **Computer Vision pipelines** for sports analytics and video understanding
- 🔁 Automate the full **MLOps lifecycle** — CI/CD, model serving, monitoring, observability
